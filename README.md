## 介绍
在别人的基础上改出来的python vim编辑器配置.

配置文件 `plugin/settings/Settings.vim`, 其他配置放在 `plugin/settings` 文件夹下。

一些vim文件:

1. `vimrc`, 标准vim文件.

2. `easy-vimrc`, 简单版.

3. `pro-vimrc`, 专业版.

4. `experimental-vimrc`, 类似专业版.

clone之后直接 link  `~/.vimrc`
